grab2web 1.0

Was ist grab2web?

Dieses Programm ist ein kleines Tool um Bilder einer Videoquelle (z.B. einer Webcam) ins Internet zu stellen. grab2web läuft dabei im Hintergrund und kann über ein Icon in der Symbolleiste jederzeit umkonfiguriert werden.

Voraussetzungen

Um grab2web nutzen zu können, müssen folgende Bedingungen erfüllt sein:

  1. DirectX 9b muss installiert sein
  2. Eine Videoquelle mit WDM Treiber muss vorhanden sein (z.B. eine Webcam oder eine Frame Grabber-Karte)
  3. FTP-Server für den Upload der Bilddaten
  4. Overlayfähige Grafikkarte

Automatisierung des Programmes

grab2web kann automatisch im Hintergrund ausgeführt werden. Dazu müssen alle nötigen Angaben zu Videoquelle und FTP-Server einmal im Programm hinterlegt worden sein (damit werden automatisch abgespeichert). Bitte beachten Sie, dass auch das Passwort für den FTP-Server unverschlüsselt in der Windows-Registrierung hinterlegt wird.
Sind die Daten einmal hinterlegt, kann das Programm mit folgenden Parametern aufgerufen werden:

  1. -h
    Dieser Parameter minimiert das Programm direkt nach dem Programmstart
  2. -r
    Dieser Parameter startet die Bildaufzeichnung direkt nach dem Programmstart
  3. -c<Ordner>
    Dieser Parameter legt eine Kopie jedes angefertigten Bildes im Ordner "<Ordner>" ab. Wenn Ordnerpfad ein Leerzeichen enthält, müssen sie diesen Parameter in Anführungszeichen angeben ("-cC:\Daten\Meine Bilder\Webcam").

Hinweise

Das Programm verwendet das JPEG-Format zur Speicherung der Bilder. Zwischenzeitlich werden die Bildaten jedoch im Windows Bitmap Format gehalten. Das Windows Bitmap Format kann nicht mehr als 24 Bit Farbtiefe verwalten. Bitte wählen Sie deshalb nur Video-Format mit einer Farbtiefe kleiner oder gleich 24 Bit, da sonst nur weiße oder schwarze Bilder abgespeichert werden.
